GAN Networks for Simulating Brain Vessel Images for Cerebrovascular Diseases Detection

Machine learning is being introduced to the diagnosis of Cerebrovascular Diseases. However, an accurate model requires a large dataset of DSA images are hard to obtain. To solve this, I created a DCGAN and a WGAN network to simulate brain blood vessel images to augment datasets. Using the binary cross entropy loss function to optimize the model over 120 epochs, DSA head images like the ones to the right were able to be simulated.

Skills used: Python, Machine Learning, SKlearn, Data Preprocessing
